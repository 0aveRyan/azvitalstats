About

Once a day, this site uses GitHub Actions to scrape the current year's spreadsheet, ingest and normalize the data and regenerates the necessary feeds. Changes to files can be tracked in the version history (look for commits from "Robot").

This site is hosted on GitHub Pages and the source code is available on GitHub under the MIT License.

Data is prepared by ADHS Bureau of Public Health Statistics
